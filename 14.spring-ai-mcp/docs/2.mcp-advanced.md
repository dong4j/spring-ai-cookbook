1.	为什么要脱离 Spring AI 看 MCP？

  - initialize / initialized 生命周期
  - tools / resources / prompts
  - transport（stdio / http / sse）

3.	从零写一个最小的 MCP Server
  - 选一个语言（例如 Java / Python）+ 官方 SDK
  - 定义一个最简单的工具（如 echo / time.now）
  - 实现 initialize 响应 + 工具列表返回 + tools.call 逻辑
  - 用命令行 / curl / 自定义 client 调一下
4.	写一个最小的 MCP Client
